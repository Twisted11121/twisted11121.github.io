Java UI Reference Guide

Quick-start snippets for common Android Studio components.

TextView & Button

The core interaction pair. Use findViewById to link the Java logic to XML.

JAVA 
TextView myText = findViewById(R.id.textView);
Button myBtn = findViewById(R.id.button);

myBtn.setOnClickListener(new View.OnClickListener() {
    @Override
    public void onClick(View v) {
        myText.setText("Button Clicked!");
    }
});

RadioButtons & Group

Used for "Select One" scenarios. Wrap buttons in a RadioGroup.

JAVA 
RadioGroup group = findViewById(R.id.radioGroup);
group.setOnCheckedChangeListener((rg, checkedId) -> {
    RadioButton rb = findViewById(checkedId);
    Log.d("Selection", "User picked: " + rb.getText());
});

SeekBar

Allows range selection (e.g., volume or brightness).

JAVA 
SeekBar sb = findViewById(R.id.seekBar);
sb.setOnSeekBarChangeListener(new SeekBar.OnSeekBarChangeListener() {
    @Override
    public void onProgressChanged(SeekBar seekBar, int progress, boolean fromUser) {
        // 'progress' holds the current value
    }
    @Override public void onStartTrackingTouch(SeekBar s) {}
    @Override public void onStopTrackingTouch(SeekBar s) {}
});

ListView & Adapter

Displays a scrollable list. The Adapter acts as the bridge between your data and the UI.

JAVA 
ListView listView = findViewById(R.id.listView);
String[] data = {"Android", "iOS", "Windows", "Linux"};

ArrayAdapter<String> adapter = new ArrayAdapter<>(this, 
    android.R.layout.simple_list_item_1, data);

listView.setAdapter(adapter);

WebView

Embeds a browser window. Requires Internet Permission in Manifest.

JAVA 
WebView web = findViewById(R.id.webView);
web.setWebViewClient(new WebViewClient()); // Stay inside app
web.getSettings().setJavaScriptEnabled(true);
web.loadUrl("https://www.google.com");

CalendarView

A simple widget for date selection.

JAVA 
CalendarView cv = findViewById(R.id.calendarView);
cv.setOnDateChangeListener((view, year, month, day) -> {
    String date = day + "/" + (month + 1) + "/" + year;
    Toast.makeText(this, date, Toast.LENGTH_SHORT).show();
});
