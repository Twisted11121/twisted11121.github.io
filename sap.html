SharedPreferences

Use this to save small amounts of primitive data (like settings or scores) that persist even after the app is closed.

Writing Data 
SharedPreferences pref = getSharedPreferences("MyPrefs", MODE_PRIVATE);
SharedPreferences.Editor editor = pref.edit();

editor.putString("username", "JohnDoe");
editor.putInt("user_level", 5);
editor.apply(); // Saves asynchronously
Reading Data 
SharedPreferences pref = getSharedPreferences("MyPrefs", MODE_PRIVATE);
String name = pref.getString("username", "Guest"); // "Guest" is the default value
int level = pref.getInt("user_level", 0);

BroadcastReceiver

Enables the app to receive intents broadcast by the system or other apps (e.g., "Battery Low").

Java Class 
public class MyReceiver extends BroadcastReceiver {
    @Override
    public void onReceive(Context context, Intent intent) {
        if (Intent.ACTION_AIRPLANE_MODE_CHANGED.equals(intent.getAction())) {
            Toast.makeText(context, "Airplane Mode Changed", Toast.LENGTH_SHORT).show();
        }
    }
}
AndroidManifest.xml 
<receiver android:name=".MyReceiver" android:exported="true">
    <intent-filter>
        <action android:name="android.intent.action.AIRPLANE_MODE" />
    </intent-filter>
</receiver>

CalendarView to String

Convert the selected date into a formatted string for display.

Java Implementation 
calendarView.setOnDateChangeListener((view, year, month, dayOfMonth) -> {
    // Month is 0-indexed: January = 0, February = 1
    String selectedDate = dayOfMonth + "/" + (month + 1) + "/" + year;
    textView.setText(selectedDate);
});

ListView & Adapter

The standard way to display scrollable data rows.

Java Implementation 
ListView lv = findViewById(R.id.myList);
String[] items = {"Item 1", "Item 2", "Item 3"};

ArrayAdapter<String> adapter = new ArrayAdapter<>(this, 
    android.R.layout.simple_list_item_1, items);

lv.setAdapter(adapter);
